  集合(合流) 
  パーティーメンバーはどこに集合すればいいかという明確な決まりはありません。基本的にはホストプレイヤーがダンジョンの入口で待っているので、ダンジョンの中に集合することが多かったです。
　ただし既にパーティーを組んでどこかで戦闘を行っている場合は、お互いに連絡をとってから頑張って集合してください。
  またホストプレイヤーのゲームが強制終了した場合は、ゲストだったプレイヤーは町の分かりやすい場所に集合するといいでしょう(特にMain Gate前)。


  会話 
　パーティーでの会話は基本的に「:p」を使って行います。この方法ならば、 マップが離れていても内容を伝えることができますし、他人に読まれることもありません。 全員が固まっている場合は、そのままチャットの内容を打ち込んでも(「:say」を使う)伝えることができますが、 あまり離れてしまうと意味がありません。

　パーティーチャットの開始に付けなくてはならない「:p」はShift＋スペースキーで入力されます。覚えておくと素早いチャットが可能になるでしょう。


  戦闘 
1.戦闘の前に
　敵が攻撃する前に、必ずどの敵を攻撃するかを知らせましょう。デフォルトのマクロではF1に設定されている「◯◯を発見しました」というメッセージをパーティーに伝えれば大丈夫です。似たような敵が複数いる場合は、メッセージを送ってすぐに叩くのではなく、メッセージを出した後で対象に近寄り攻撃するくらいの間が欲しいものです。
  どの敵を倒すかは誰が決めてもいいのですが、基本的にパーティーリーダーもしくは(後述する)前衛タイプのキャラクターがいいです。強すぎる敵を攻撃したり、複数を誘き寄せたりするのはやめたほうがいいかもしれません。

2.前衛、後衛
  戦闘時は敵を中心に取り囲んだ状態で戦うことが多いので、前衛や後衛という言い方は間違っていますが、前衛が敵の壁になり攻撃をする人、後衛が魔法で味方を援護したり敵を攻撃したりする人です。
  始めに敵に対し攻撃を仕掛ける(釣る)のは、前衛型の回避力が高いプレイヤーか防御力の高いプレイヤーの役割です。ヒーラーのような防御力が低く味方の援護を専門とするクラスが先制攻撃をすると、パーティーの全滅につながることもあります。

3.休憩
  HPやMPが少なくなってきたら、遠慮なく休憩をしましょう。パーティーメンバーもステータスが分かっているので、文句を言われることはまずありません。特にHPもMPも1/4前後になったら絶対に休憩するべきです。
  また早めに休憩が終ったプレイヤーは、パーティー全員のステータスが回復するまで周囲の警戒を怠らないようにしてください。時々aggro の敵が戦闘状態になり襲って来ることがありますので、そのような場合はすぐに報告しクラスにかかわらず取り合えず敵の相手をしてください。敵が襲ってきたときに素早く対処するため、休憩が終ったら立っている方が無難です。


  宝箱 
　宝箱を見つけた場合は、基本的に発見者が開けることになっています。 もし釘ぬき等がなく開けられない場合は、開けるためのアイテムを持っている人に権利が移ります。 これは明確な基準ではないので、最終的には譲り合で何とかしてください。

　譲り合いといっても、宝箱に入っているお金はパーティーで山分けですし、おそらく中のアイテムも戦利品として分配されるので、開けるときの快感を味わうかどうかの選択なのですが。

  移動 
　パーティーでの移動は、パーティーリーダーが先頭となって移動するのが基本です。 そのためパーティーリーダーはある程度そのマップについて詳しくなければなりません。
　また長い距離を移動する(リドニアからペンタウァへ行脚する)などはメンバー全員が地形について詳しいほうがい いです。本当はマップよりも敵がaggro(アクティブ)かどうかを知っているほうが重要なのですが。


  ロット 
　パーティーでゲームを進める場合、入手したアイテムは「戦利品」として全員が共有するアイテム袋(?)の中に収集されます。共有といってもアイテム袋の中身は自由に利用できるのではなく、「ロット」 をして自分のものにしない限り利用できません。
　ロットをするにはメニュー1ページ目もしくは2ページ目の「戦利品」を選び、欲しいアイテムを選択して 「ロット」を選んでください。ロットの原理はよくわかりませんが、ロットが成立するとロットインしたアイテムが自分の手元に入ります。
　ロットを頻繁に行うパーティーに参加した場合、いらないアイテムは「戦利品」のウィンドウで「パス」することも必要です。

　なお、ロットの原理は1～100までの数が書かれているサイコロを、1人1回ずつ振り、一番大きい目が出た人にアイテムを渡すという形になっています。一人がロットを行った場合、他のメンバーが一定時間内にロットを行わない限り、始めにロットした人のものとなるので注意してください。またロットは一つのアイテムについて一度しかできません。

  経験値とお金 
　パーティーでプレーする場合、倒す敵のレベルを間違えると入ってくる経験値(ソーサリアンオンラインの場合は、ステータスの上昇する頻度)は極端に低くなります。同レベルの敵を倒しても、全く経験値が入りません。
　また、敵を倒したときに得られるお金は、パーティーで山分けとなります。
　クラスの組み合わせにもよりますが、ランク1の70以降はパーティーを組むことをおすすめします。

　パーティーでプレーしている場合、経験値の量はパーティーメンバーで最もクラスレベルが高い人に依存します。怪しい数式表現を使いますが、パーティーメンバーで最もクラスレベルが高いプレイヤーをA とし、 B = { ( Aのランク - 1 ) * 100 + Aのクラスレベル } / 10 とします。
　このとき、ソロプレーと同じ経験値を全員が得るには、 B + [A以外のパーティーメンバーの数] + 1 以上のレベルの敵と戦うといいみたいです。ちなみに、他のパーティーメンバーの経験値もA に入るものと同じ程度です。そのため、ランク2 ばかりのパーティーに一人だけランク1 のプレイヤーがいてもその人が急激にに育つことはありません。

　具体的な例を挙げると、A がランク1 のレベル70 の時、A以外パーティーメンバーの数が・・・
1人 ・・・ Lv.9 以上
2人 ・・・ Lv.10 以上
3人 ・・・ Lv.11 以上

　Aがランク2 でレベル15 の時、A以外のパーティーメンバーの数が・・・
1人 ・・・ Lv.12 以上
2人 ・・・ Lv.13 以上
3人 ・・・ Lv.14 以上

　かなりまれなケースですが、パーティープレイヤーが別のマップにいる場合、そのプレイヤーには経験値が分配されません。例えば、4人パーティーで1人だけが別のマップにいるとします。このとき3人が一体の敵を倒しても、経験値はそのマップ上にいる3人にのみ分配されます。お金の場合は確認していませんが、おそらく3人に分配されるはずです。


　以下の情報は不確定な厳密な検証を行っていませんが、恐らく正しいはずです。
　頭の回転が速い人ならば、すぐに気づくでしょう。あえてパーティーを組まない状態で強いプレイヤーに敵を弱らせてもらい、自分でとどめを刺せば経験値が簡単に稼げるだろうと。
　結論からいうとこの方法はソーサリアンオンラインでは通用しません。上の適正レベルの式は厳密にいうと間違っており、正確には以下のようになります。太字、赤字の部分を比較してみてください。

　同じマップにいる一番強いプレイヤーのクラスレベルをAとし、B = { (Aのクラスランク - 1) * 100 + Aのクラスレベル } / 10 とします。
　このときマップにいるプレイヤー達が倒す敵の適正レベルは、B + 同じマップにいるプレイヤーの人数 となります。

　つまりパーティーを組まずとも、同じマップにいるだけで干渉してしまうのです。
