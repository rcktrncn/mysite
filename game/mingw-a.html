ここから、「Mingw-4.1.0.exe」をダウンロードする。個人的には、「北陸先端技術大学院大学」の828Kbからのダウンロードがおすすめ。

取りあえず、インストールする。インストールのフォルダはデフォルトの「C:\mingw\」にしてください。

スタートメニュー → プログラム → アクセサリ → コマンドプロンプト

「 cd .. 」を何度か打ち込んで、

c:\>

と表示させる。

「 cd mingw\mingw 」

「 path ../bin;%path% 」

を入力。
これで、gccが使えるようになる。命令のやり方は、WS室のものと同じ

例：foo.cをコンパイルして、foo.exeを作成する
    gcc -o foo.exe foo.c

例：foo.c、foo2.cをコンパイルして、foo.exeを作成する
    gcc -o foo.exe foo.c foo2.c

例：foo.exeを実行する
    foo

その他、Windowsで覚えておくといいコマンド
cd
 指定したディレクトリーへ移動。WS室のあれの命令と同じ
dir
 現在のディレクトリーにあるファイルを表示
del
 指定したファイルを削除
mkdir
 ディレクトリーの作成
path
 pathを通す
cls
 画面をキレイにする
