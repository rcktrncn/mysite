更新にあたって (ただの日記かも)

「時代が変わりますね」
「らんの眼」を始めた当初からずっとお世話になっていたジオシティーズが終了ですね。
さみしいですが、これも時代の流れです。
そのため WebCrow に一本化しようかと思ったのですが、サービス内容が変わったので、 GitHub に移動することにしました。
まさかプログラム開発で使うより先に、ウェブサイトのためにここを利用するとは思いませんでした。
取りあえず移動はしましたが、もしかしたらリンク切れがたくさんあるかもしれません。
