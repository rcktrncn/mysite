無料で使える RPA の taskt の使い方を紹介しています

Create Excel Application

Excel を起動します。当然ですが、パソコンに Excel がインストールされている必要があります。

taskt では Excel ごとに「インスタンス」という名前を付けて管理することで、複数の Excel のファイルを同時に操作することができます。

そして、taskt のほかの Excel 関係の命令は、すべてインスタンスを指定することになりますので、名前を忘れないようにしてください。

項目 意味
Please Enter the instance name インスタンス名を入力します
インスタンス名は「ex」や「excel」など重複がなければ良いです
Comment Field (Optional) コメントを入力します
オプションなので入力しなくても良いです

 

高度な話になりますが、Create Excel Application 命令が実行されると、Excel が起動します。

Excel が完全に起動し終わるまで、スクリプト中の次の命令は実行されません。

Create Excel Application 命令終了後、Excel はファイルを開いておらず、新規でワークブックを作った状態でもないので、いきなりデータの入力などをすることはできません。

そのため、Create Excel Application 命令の後に、Excel ファイルを開く Open Workbook 命令や、新規ワークブックを作る Add Workbook 命令を実行してください。
