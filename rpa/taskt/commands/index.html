無料で使える RPA の taskt の使い方を紹介しています

コマンド一覧

taskt のコマンドを 1 つずつ説明する予定です。

使い方が分からないものは説明しません。

API Commands (API を操作する)

Windows API とか操作するコマンドだと思います。

  • Excecute DLL (DLL を呼び出す)
  • Excecute REST API
  • HTTP Request (HTTP リクエストを送る)
  • HTTP Result Query

Data Commands (文字や日付などのデータを操作する)

  • Date Calculation (日付の計算をする)
  • Format Data (日付の書式を変更する)
  • Get List Count (リストの要素数を取得する、バージョン 3.5.0.0 で追加)
  • Get List Item (リストの個別の要素を取得する、バージョン 3.5.0.0 で追加)
  • Get Word Count (単語の数を数える)
  • Get Word Length (文字数を数える)
  • Log Data
  • Math Calculation (数値計算をする、バージョン 3.4.0.0 で追加)
  • Modify Variable (大文字・小文字・Base64 変換など)
  • Parse Dataset Row
  • Parse JSON Array (JSON の配列を taskt のリストに変換する)
  • Parse JSON Object (JSON をパースする)
  • Parse JSON Item (JSON から値を取得する、バージョン 3.4.0.0 か 3.5.0.0 で追加)
  • Parse JSON Model (バージョン 3.4.0.0 で追加)
  • PDF Extraction
  • RegEx Extraction (正規表現)
  • Replace Text (文字を置換する)
  • Split Text (文字を分割する)
  • Substring (文字の一部を取り出す)
  • Text Extraction (文字を検索し、特定の範囲を取り出す)

Database Commands (データベースを操作する)

データベースを操作できるのではないでしょうか……。

  • Define Database Connection
  • Execute Database Query

DataTable Commands

よくわかりません。バージョン 3.4.0.0 で追加されました。

  • Add DataRow
  • Create DataTable
  • Filter DataTable
  • Get DataRow (バージョン 3.5.0.0 で追加)
  • Get DataRow Count (バージョン 3.5.0.0 で追加)
  • Get DataRow Value (バージョン 3.5.0.0 で追加)
  • Remove DataRow
  • Write DataRow Value (バージョン 3.5.0.0 で追加)

Dictionary Commands

よくわかりません。バージョン 3.4.0.0 で追加されました。

  • Add Dictionary Item
  • Create Dictionary
  • Get Dictionary Item
  • Load Dictionary

Engine Commands (taskt のロボットに関する処理)

  • Error Handling (エラーが発生したときの処理)
  • Get BotStore Data
  • Pause Script (次の命令まで一定時間待つ)
  • Set Engine Delay (命令と命令の間隔を変える)
  • Set Engine Preference (自動計算の設定を変更する、バージョン 3.4.0.0 で追加)
  • Show Engine Context
  • Stopwatch (時間を測る)
  • Upload BotStore Data

Error Handling Commands (例外処理、エラー発生時の処理)

プログラマー的な説明だと例外処理です。

  • Catch Exception (catch)
  • End Try (例外処理の終了)
  • Finally (finally)
  • Throw Exception (例外を発生させる?、バージョン 3.5.0.0 で追加)
  • Try (例外処理の開始)

Excel Commands (Excel を操作する)

Excel を操作するときの命令です。

  • Activate Sheet (シートを選択する)
  • Add Workbook (ワークブックを追加する)
  • Append Cell (作業中のセルの 1 つ下に書き込む、バージョン 3.4.0.0 で追加)
  • Append Row (バージョン 3.4.0.0 で追加)
  • Close Excel Application (Excel を終了する)
  • Create Dataset (データセットを作る?)
  • Create Excel Application (Excel を実行する)
  • Delete Cell (セルの中身を消す)
  • Delete Row (行を消す)
  • Get Cell (セルの値を取得する)
  • Get Last Row Index (最後の行が何行目か調べる)
  • Get Range (指定した範囲の値をまとめて取得する、バージョン 3.4.0.0 で追加)
  • Get Range As Datatable (バージョン 3.4.0.0 で追加)
  • Go To Cell (指定したセルにカーソルを移動する)
  • Open Workbook (Excel ファイルを開く)
  • Run Macro (マクロを実行する)
  • Save Workbook (Excel ファイルを保存する)
  • Save Workbook As (Excel ファイルを名前を付けて保存する)
  • Set Cell (セルに値を書き込む)
  • Split Range By Column (バージョン 3.5.0.0 で追加)
  • Write Range (バージョン 3.4.0.0 で追加)
  • Write Row (バージョン 3.5.0.0 で追加)

File Operation Commands (ファイルを操作する)

ファイルの移動、コピー、削除などを行います。

  • Delete File (ファイルを削除する)
  • File Extraction (バージョン 3.5.0.0 で追加)
  • Get Files (バージョン 3.5.0.0 で追加)
  • Move/Copy File (ファイルの移動、コピーをする)
  • Rename File (ファイル名を変更する)
  • Wait For File (ファイルが存在するまで一定時間待つ)

Folder Operation Commands (フォルダーを操作する)

フォルダーの作成、削除などを行います。バージョン 3.5.0.0 で追加されました。

  • Create Folder (フォルダーを作成する)
  • Delete Folder (フォルダーを削除する)
  • Get Folders
  • Move/Copy Folder (フォルダーの移動、コピーをする)
  • Rename Folder (フォルダーの名前を変更する)

IE Browser Commands (Internet Explorer を操作する)

Internet Explorer を操作する命令だと思います。

  • Close Browser (Internet Explorer を閉じる)
  • Create Browser (Internet Explorer を実行する)
  • Element Action (ウェブページに対する操作)
  • Find Browser
  • Navigate to URL (指定したURL へ移動)

If Commands (if 文、条件分岐)

プログラミング的な説明だと if 文、条件分岐です。

  • Begin If (if 開始)
  • Begin Multi If (複数条件の if、バージョン 3.4.0.0 で追加)
  • Else (else)
  • End If (if 終了)

Image Commands (画像関係の処理)

画像認識やスクリーンショットの命令です。

  • Image Recognition (画像認識)
  • Perform OCR
  • Take Screenshot (スクリーンショットを撮る)

Input Commands (キーボードやマウスの操作)

キーボード操作や、マウス操作の命令です。

  • Prompt for HTML Input
  • Prompt for Input (入力画面を表示する)
  • Send Advanced Keystrokes (高度なキーボード操作)
  • Send Keystrokes (キーボード操作)
  • Send Mouse Click (マウスのクリックなどの操作)
  • Send Mouse Move (マウスカーソルを動かす操作)
  • UI Automation (画面の要素を認識して処理する)

Loop Commands (繰り返し、ループの処理)

プログラミング的な説明だと、ループです。繰り返し何かを実行するときの命令です。

  • Begin Loop (バージョン 3.5.0.0 で追加)
  • Begin Multi Loop (バージョン 3.5.0.0 で追加)
  • End Loop (ループ終了)
  • Exit Loop (ループを強制的に抜ける、break)
  • Loop Continuously (無限ループ)
  • Loop Excel Dataset (Excel のデータセットを見ながらループを繰り返す、バージョン 3.4.0.0 で廃止)
  • Loop List (配列・リストの先頭から末尾までのループ、for-each、for-in)
  • Loop Number Of Times (指定した回数繰り返すループ)
  • Next Loop

Misc Commands (その他、ほかのところに分類できない命令)

いろいろな命令です。

  • Add Code Comment (コメント、説明を挿入する)
  • Encryption Command (バージョン 3.4.0.0 で追加)
  • Get Clipboard Text (クリップボードにある文字を取得する)
  • Send SMTP Email (SMTP でメールを送る)
  • Sequence Command (シーケンスを作る、命令をまとめる)
  • Set Clipboard Text (クリップボードに文字を記録する)
  • Show Message (メッセージを表示する、ダイアログを表示する)

NLG Commands

よくわかりません。

  • Create NLG Instance
  • Generate NLG Phrase
  • Set NLG Parameter

Outlook Commands

Outlook を操作する命令だと思います。バージョン 3.4.0.0 で追加されました。

  • Delete Outlook Emails (Outlook のメールを削除する?、バージョン 3.5.0.0 で追加)
  • Forward Outlook Emails (Outlook のメールを転送する?、バージョン 3.5.0.0 で追加)
  • Get Outlook Emails (バージョン 3.5.0.0 で追加)
  • Move/Copy Outlook Emails (Outlook のメールの移動・コピーをする?、バージョン 3.5.0.0 で追加)
  • Reply To Outlook Emails (Oulook のメールへ返信する?、バージョン 3.5.0.0 で追加)
  • Send Outlook Email (Outlook でメールを送信する?)

Programs/Process Commands (プログラム・ソフトウェアを操作する命令)

ほかのソフト (プログラム) を操作するための命令です。

  • Run Custom Code (C# のソースコードを実行する)
  • Run Script (バッチファイルを実行する?)
  • Run Process (ソフト・プログラムを実行する)
  • Stop Process (ソフト・プログラムを終了する)

Regex Commands

よくわかりません。名前からして正規表現に関するものだと思います。バージョン 3.4.0.0 で追加されました。

  • Get Regex Matches

Remote Commands

よくわかりません。バージョン 3.3.0.0 で追加されました。

  • Remote API
  • Remote Task

System Commands (OS・パソコン本体に関係する命令)

OS (Windows) の情報や、パソコンの情報を取得したりできます。

  • Environment Variable (OS の環境変数を取得する)
  • Launch Remote Desktop (リモート・デスクトップ を実行する)
  • OS Variable (Windows が持っている情報を取得する)
  • System Action (Windows の再起動、終了などをする)

Task Commands

よくわかりません。

  • Run Task
  • Stop Current Task

Text File Commands (テキストファイルを操作する命令)

テキストファイル (メモ帳などで作れるファイル) を操作する命令です。

  • Read Text File (テキストファイルを開く、読み込む)
  • Write Text File (テキストファイルを保存する)

Variable Commands (変数に関する命令)

変数を操作するときの命令です。

  • Add To Variable (リスト・配列に要素を追加する)
  • New Variable (新しい変数を作る、変数の宣言)
  • Set Variable (変数に値を入れる、値の代入)
  • Set Variable Index (リスト・配列の要素番号を操作する、リスト・配列のインデックス操作)

Web Browser Commands (ウェブブラウザーを操作する命令)

ウェブブラウザーを操作する命令です。バージョン 3.2.0.0 の時点で Chrome と Internet Explorer に対応しているみたいです。

別途 Chrome をパソコンにインストールしてください。

Selenium を使っているので、検索エンジンで調べるときに、taskt で調べるよりも、Selenium で調べた方が有用な情報が見つかるかもしれません。

  • Close Browser (ウェブブラウザーを終了する)
  • Create Browser (ウェブブラウザーを実行する)
  • Element Action (ウェブページの要素に対する操作)
  • Execute Script (JavaScript を実行する)
  • Get Browser Info (ウェブブラウザーが持っている情報を取得する)
  • Navigate Back (前のページに戻る)
  • Navigate Forwared (先のページに進む)
  • Navigate to URL (指定した URL のページへ移動)
  • Refresh (ウェブページを再読み込みする)
  • Switch Browser Frame (ウェブページ中の対象フレームを変更する、バージョン 3.3.0.0 で追加)
  • Switch Browser Window

Window Commands (ソフト・画面に対する操作)

ソフト (プログラム) の画面に対する操作ができます。

  • Activate Window (ソフトをアクティブにする、ソフトの画面を一番前にする)
  • Close Window (ソフトを閉じる)
  • Move Window (ソフトの画面を移動する)
  • Resize Window (ソフトの画面の大きさを変更する)
  • Set Window State (ソフトの最大化、最小化をする)
  • Wait For Window To Exist (ソフトの画面が表示されるまで一定時間待つ)

Word Commands

Word を操作するコマンドです。バージョン 3.5.0.0 で追加されました。

  • Add Document
  • Append DataTable
  • Append Image (画像を挿入する?)
  • Append Text (文字を挿入する?)
  • Close Word Application (Word を閉じる)
  • Create Word Application (Word を開く)
  • Export To PDF (PDF 形式でファイルを出力する?)
  • Open Document (Word のファイルを開く)
  • Read Document
  • Replace Text
  • Save Document (Word のファイルを上書き保存する)
  • Save Document As (Word のファイルを名前を付けて保存する)
