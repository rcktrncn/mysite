無料で使える RPA の taskt の使い方を紹介しています

taskt とは？

taskt とは、無料で使える RPA です。

もっと厳密な説明をすると、オープンソースで開発されている無料の RPA です。ライセンスとしては Apache Lisence Version 2.0 らしいです。

個人での利用はもちろん、商用での利用も可能です。

taskt で何ができるか

taskt は無料で使える RPA ですが、いろいろなことができます。

例えば、下のような感じです。取りあえず使ってみたい方は、簡単な使い方をご覧ください。

  • Excel の作業を自動化する
  • ウェブサイトからの情報を取得してファイルとして保存する
  • たくさんのファイルを開き、同じ操作をさせる
  • キーボードやマウスの操作の自動化
  • Outlook でメールを送信する (バージョン 3.4.0.0 以降)
  • Word で文章を作る (バージョン 3.5.0.0 以降)
  • などなど

もっと詳しく知りたければ、コマンド一覧やサンプルを眺めるといいかもしれません。

taskt の利点と欠点は？

taskt は無料で使えますが、ほかの RPA と比べての利点・欠点があります。

利点はこんな感じです。

  • 無料で使える (会社の規模によらず無料)
  • 編集画面がそんなに重たくない
  • 工夫次第でだいたいのことが自動化できる
  • 作ったデータが XML 形式なので扱いやすい (かも)

欠点はこんな感じです。

  • サポート体制がほぼない (英語が使えればチャットで質問できるらしいです)
  • マニュアル・説明書がほぼない (英語が読めれば説明が見れます)
  • 画面の表示が英語 (バージョン 3.2.0.0 時点)
  • いかにもプログラミングっぽい画面で、ビジュアル的にやや分かりづらい

taskt の実行環境

taskt の実行環境について、バージョン 3.3.0.0 で正式な情報が公開されました。

Windows 7 以降なら動作するのですが、.NET Framework 4.8 がインストールされていないと taskt が動かせないので注意してください。詳細は、インストールと実行を見てください。

推奨環境

OS Windows 10
Windows Server 2012
Windows Server 2016
メモリー 4 GB
ハードディスク 1 GB
CPU 2 コア以上の CPU
.NET Framework 4.8 が必須

 

最低環境

OS Windows 7
Windows 8.1
Windows 10
Windows Server 2012
Windows Server 2016
メモリー 1 GB
ハードディスク 200 MB
CPU 1 コア以上の CPU (仮想 CPU でもよい)
.NET Framework 4.8 が必須
