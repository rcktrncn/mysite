バージョン履歴 v3.x.x.x

v3.3.0.0 (2019/12/07)

  • Switch Browser Frame 命令が使えるようになった
  • API 越しに個別の命令を使えるようになった?
  • トレイアイコンを削除
  • Excecute Script 命令で JavaScript からの戻り値を取得できるようになった (Selenium のみ)
  • TCP リスナーで IP のチェックとホワイトリストが使えるようになった?
  • 説明の追加?
  • 実行を待つコマンドを更新した?
  • API の実行結果が返ってくるで待つようにした?
  • (タイトルなどのグラデーションの向きが逆になった?)

v3.2.2.0 (2019/10/06)

  • ロボットを実行するとき、実行中 (青)、エラー (赤)、正常終了 (緑) に変わるようにした
  • Settings Manager に Advanced Debug Log の設定を追加した
  • HTTP Request 命令で現在のユーザーで実行するかどうかの選択をドロップダウンリストでできるようにした
  • HTTP Request 命令で標準ユーザーの資格を追加した?
  • If 命令の表示を変えた
  • User Input 命令で標準の値についての不具合を修正した?

v3.2.1.0 (2019/09/03)

  • メールの処理を追加
  • taskt の実行ファイルのある場所を取得できるようにした

v3.2.0.0 (2019/08/02)

  • Selenium のインスタンスを変更できるようにした?
  • Browser Info 命令が使えるようになった
  • REST API 命令を強化した
  • Show Engine Context 命令が使えるようになった
  • データベースの初期化に対応?
  • クエリーパラメーターに対応?
  • 文字列と暗号化されたパスワードに対応?
  • Engine Context Viewr が自動的に閉じる処理を追加
  • Try, Catch, Finally, End Try 命令が使えるようになった

v3.1.0.0 Preview Build (2019/08/01)

  • 不明

v3.0.0.0 (2019/07/14)

  • NLG が使えるようになった (SimpleNLG を使っている)
  • IE Automation 命令がまた使えるようになった
  • リストにインデックスを指定できるようになった
  • 変数から出力できるプロパティーを複数追加した
  • .NET Framework 4.8 にした
  • 複数の不具合を修正
