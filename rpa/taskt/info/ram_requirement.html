無料で使える RPA の taskt の使い方を紹介しています

公式の推奨環境は正しいのか

taskt の公式の情報では、推奨環境は下の表のように記載されています。

OS Windows 10
Windows Server 2012
Windows Server 2016
メモリー 4 GB
ハードディスク 1 GB
CPU 2 コア CPU
.NET Framework 4.8

 

この情報ですが、おおむね正しいのです。

しかし、唯一メモリー 4 GB だけは怪しい面があります。

taskt のスクリプトを開くときと、スクリプトを実行するとき、一時的ですがかなりのメモリーを消費します。スクリプトの長さや、パソコンのメモリーの空き容量にもよりますが、2 GB 以上を消費することもあります。

Windows 10 の場合、何もしていなくても 2 GB くらいはメモリーを消費している気がするので、メモリーが 4 GB しかない場合、長いスクリプトを開いたり実行しようとすると、一時的にパソコンの動作が非常に遅くなります。

そのため、taskt のスクリプト開発などでガンガン使うパソコンならば、快適に利用するために、メモリーは 8 GB くらいはあった方が良いと思います。

ただし、単にスクリプトを実行するだけのパソコンならば、一時的に遅くなるだけなので、気にする必要はあまりないかもしれません。
