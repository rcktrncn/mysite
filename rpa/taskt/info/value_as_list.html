普通の変数をリストとして扱うとどうなるか

本来はやってはいけないのですが、普通の値が入っている変数をリストとして扱うとどうなるかを調べました。

変数 var には RED という文字が入っているものとします。

書き方 取得できる値
{var.First} RED
{var.Last} RED
{var.Index} RED
{var.Count} RED
{var.Type} BASIC

どうやら、普通の変数をリストとして扱っても無視されます。

また、リストの要素数を取得する Get List Count 命令や、リストの指定した要素を取得する Get List Item 命令を実行しようとするとエラーになります。
