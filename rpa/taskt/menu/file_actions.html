無料で使える RPA の taskt の使い方を紹介しています

File Actions

メニューの File Actions は、スクリプトファイルを開いたり、保存したりできます。

各項目の説明

項目 意味
New File
 新しいスクリプトファイルを作ります
Open Existing File
 スクリプトファイルを開きます
Import File
 現在編集中のスクリプトファイルに、ほかのスクリプトをインポート (追加・挿入) します
Save
 スクリプトファイルを保存します
Save As
 スクリプトファイルを名前を付けて保存します
Restart Application
 taskt を再起動します
Close Application
 taskt を終了します

バージョン 3.5.0.0 現在、taskt では新しいスクリプトファイルを作る Ctrl+N、スクリプトファイルを開く Ctrl+O、保存する Ctrl+S といった一般的なショートカットキーは使えません。
