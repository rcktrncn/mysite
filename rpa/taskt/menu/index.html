その他機能

ここでは、主に taskt の上のメニューの項目を説明します。

  • File Actions (スクリプトファイルを開く、保存する)
  • Options
    • Variable Manager (変数の管理)
    • Settings Manager (taskt の設定)
      • Application
      • Automation Engine
      • Server
      • Local Listener
    • Show Search Bar
  • Script Action
    • Record (操作をスクリプトとして記録する)
    • Schedule (スクリプトをタスクスケジューラーに登録)
    • Run (スクリプトを実行する)
  • Save and Run (スクリプトを保存して実行する)
