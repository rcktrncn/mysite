無料で使える RPA の taskt の使い方を紹介しています

変数を用意する

変数を用意する (宣言する) には、Variable Manager を使います。

New Variable 命令でもできるのですが、ここでは扱いません。

メニューの Options から Variable Manager をクリックします。

Variables という画面が表示されるので、 をクリックします。

Add Variable という画面が表示されるので、上の入力エリア (Define Variable Name) に変数の名前を入力します。

変数の初期値を指定したければ、下の入力エリア (Define Variable Default Value) に初期値を入力します。

一般的なプログラミング言語のように、文字列を指定するとき " " で囲む必要はありません。

OK をクリックします。

今用意した変数 var が追加されます。

もし、初期値を指定した場合、Default Value に初期値が表示されています。
