特定の命令を一時的に無効化する

注意　ここの内容はバージョン 3.2.0.0 時点の内容です。バージョン 3.5.0.0 時点で右クリックしたときのメニューの項目が増えています。ただし内容に大きな違いはありません。

スクリプトを編集していると、特定の命令だけを一時的に無効化したいときがあると思います。

たとえば、下の選択中の Send Keystrokes [Send 'C' to …… を無効化したいときは、

選択中の命令の上で右クリックし、出てきたメニューから Disable Selected Code をクリックします。

残念ながら、バージョン 3.2.0.0 ではこの操作をした段階では選択中の命令の見た目に変化がありません。

しかし、別の命令を選択すると、今 Disable Selected Code を選択した Send Keystrokes [Send 'C' to …… は緑色に変わっており、スクリプト実行中にこの命令は無視されます。

無効化を解除するには、再度選択して右クリックし、出てきたメニューから Enable Selected Code をクリックします。

複数の命令をまとめて選択した後、一括で無効化・有効化することも可能です。
