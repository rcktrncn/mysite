無料で使える RPA の taskt の使い方を紹介しています

サンプル

taskt のスクリプトのサンプルです。

変数を使う

  • 変数の代入、変数の参照、Get Clipboard Text (スクリプトをダウンロード)
  • リスト・配列の代入、リスト・配列の参照 (スクリプトをダウンロード)
    Add To Variable、Set Variable Index

ソフト (プログラム) の実行

  • Run Process、Send Keystrokes (スクリプトをダウンロード)

キーボードの入力

  • Run Process、Send Keystrokes (スクリプトをダウンロード)
  • Loop List、Prompt for Input (スクリプトをダウンロード)

Excel の操作

  • Excel ファイルの作成・編集 (スクリプトをダウンロード)
    Create Excel Application、Add Worksheet、Activate Sheet、Set Cell、Close Excel Application
  • Excel ファイルを開く・編集 (スクリプトをダウンロード、Excel ファイルをダウンロード)
    Create Excel Application、Open Workbook、Get Last Row Index、Get Cell

ウェブサイトの操作

  • ウェブサイトの情報を取得 (スクリプトをダウンロード)
    Create Browser、Navigate to URL、Element Action、Close Browser
  • ウェブサイトを操作、ウェブサイトに情報を入力 (スクリプトをダウンロード)
    Create Browser、Navigate to URL、Element Action、Close Browser

繰り返し・ループ

  • Loop Number of Times、ループのカウンター (スクリプトをダウンロード)
  • Loop List、Prompt for Input (スクリプトをダウンロード)

クリップボードの操作

  • 変数の代入、変数の参照、Get Clipboard Text (スクリプトをダウンロード)

テキストファイルの操作

  • Read Text File、Text Split (スクリプトをダウンロード)

ウインドウ・画面の操作

  • ウインドウ・画面の操作 (スクリプトをダウンロード)
    Activate Window、Move Window、Set Window State
  • Activate Window、アクティブなウィンドウ名の取得 (スクリプトをダウンロード)

日付・時刻

  • Date Calculation、現在の日付を取得 (スクリプトをダウンロード)
  • Stopwatch (スクリプトをダウンロード)

ファイル操作

  • Move/Copy File、Rename File (スクリプトをダウンロード)

ネタ

  • Word で自動的に文章作成 (Word 2010 用、Word 2016, 2019 用)
  • Excel で自動的にグラフ作成 (スクリプトをダウンロード)
  • ペイントを操作する (スクリプトをダウンロード)
