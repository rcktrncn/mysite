無料で使える RPA の taskt の使い方を紹介しています

Read Text File、Split Text

ここの内容はバージョン 3.3.0.0 までの変数を意味する囲いが [ ] であるときの内容です。そのため、最新のバージョンには対応していない部分があります。

  1. My Scripts フォルダーにある sample_text.txt を開き、変数 [textFile] へ入れる
  2. 変数 [textFile] の内容を行ごとに区切り、リスト [spt] に入れる
  3. リスト [spt] を順番に表示

スクリプトのダウンロード、テキストファイルのダウンロード

text_file_open.xml と sample_text.txt は My Scripts フォルダーに入れてください。

変数 [spt]、[textFile] の宣言

sample_text.txt を開き、変数 [txtFile] へ入れる

変数 [textFile] の値を行ごとに区切り、リスト [spt] に入れる

リスト [spt] のループ

リスト [spt] の値を順番に表示
